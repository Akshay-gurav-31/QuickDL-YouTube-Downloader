## Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/QuickDL.git
    cd QuickDL
    ```

2.  **Backend Setup**:
    ```bash
    cd backend
    pip install -r requirements.txt
    python app.py
    ```

3.  **Usage**:
    Open `http://localhost:5000` in your browser.
